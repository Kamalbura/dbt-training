================================================================================
            DBT SEED vs MANUAL UPLOAD - EXPLAINED
================================================================================

YOUR QUESTION: "Does dbt seed send datasets to the cloud like BigQuery?"

SHORT ANSWER: YES! dbt seed loads CSV files from your seeds/ folder into BigQuery.


================================================================================
                           DBT SEED EXPLAINED
================================================================================

WHAT IS DBT SEED?
-----------------
dbt seed is a command that loads CSV files into your data warehouse.

Command: `dbt seed`

What it does:
1. Looks in your seeds/ folder
2. Finds CSV files (like raw_customers.csv)
3. Creates tables in BigQuery
4. Loads the CSV data into those tables

Example:
If you have seeds/countries.csv, running `dbt seed` creates:
  `saras-bigquery.dbt_training.countries`


WHEN TO USE DBT SEED:
---------------------
✅ Reference/lookup data (countries, categories)
✅ Static data that rarely changes
✅ Small datasets (< 1GB)
✅ Data you want version-controlled in git

❌ Large datasets (use BigQuery console or ETL tools)
❌ Data that updates frequently
❌ Your Shopify orders data (already in BigQuery!)


================================================================================
                           MANUAL UPLOAD vs DBT SEED
================================================================================

BOTH put data in BigQuery, but they're DIFFERENT tools for DIFFERENT purposes:

┌─────────────────┬─────────────────────┬─────────────────────┐
│                 │ MANUAL UPLOAD       │ DBT SEED            │
├─────────────────┼─────────────────────┼─────────────────────┤
│ Method          │ BigQuery Console    │ dbt seed command    │
│                 │ or bq load          │                     │
├─────────────────┼─────────────────────┼─────────────────────┤
│ Data Source     │ Any file type       │ CSV files only      │
│                 │ (JSONL, CSV, etc.)  │                     │
├─────────────────┼─────────────────────┼─────────────────────┤
│ File Location   │ Local or Cloud      │ seeds/ folder       │
├─────────────────┼─────────────────────┼─────────────────────┤
│ Version Control │ Manual              │ Git-tracked         │
├─────────────────┼─────────────────────┼─────────────────────┤
│ Automation      │ One-time            │ Repeatable          │
├─────────────────┼─────────────────────┼─────────────────────┤
│ Use Case        │ Raw data loading    │ Reference data      │
└─────────────────┴─────────────────────┴─────────────────────┘
